Übung Visitor
=============

Die Klassen Waschbecken und Fernseher sollen sowohl reparierbar als auch reinigbar werden.

Das gesamte Wissen über die Reinigung soll in der Klasse Putzfrau gesammelt werden.
Das gesamte Wissen über die Reparatur soll in der Klasse Handwerker gesammelt werden.

Erweitern Sie die Klassen Waschbecken und Ferseher so, dass diese (von einer Putzfrau) besucht werden können.
Hinterlegen Sie in der Klasse Putzfrau das gesammelte Wissen zum Putzen von Klassen.
Ändern Sie die Klasse Handwerker so, dass dieser bei seinem Besuch den Fernseher und Waschbecken reparieren kann.

Optional:
Erstellen Sie eine Klasse Versicherungsvertreter, der bei seinem Besuch alles versichert.
